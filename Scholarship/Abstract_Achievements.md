# 主要学术成果摘要

本人在本科阶段曾参与多项医学工程与智能手术机器人相关科研项目，在医学图像处理、图像三维重建与手术路径规划等方面积累了一定工作经验，锻炼了良好的科研基础与工程实践能力。相关成果主要体现在以下两个科研项目中。

## 一、科研项目与技术成果概述

### 心血管及外周血管智能介入机器人及其自主导航方法研究

该项目依托深圳市医学研究专项资金，针对机器人辅助血管介入（Robot-Assisted Vascular Intervention, RAVI）领域中“高度依赖医生经验、自动化与智能化程度不足”的关键问题，围绕具身智能（Embodied AI）框架，开展血管介入机器人自主学习与智能决策方法研究。项目从自动化、精准性与实时性需求出发，结合多任务深度强化学习、多模态感知融合及多尺度血管建模方法，构建面向复杂临床环境的智能决策与控制体系。

在该项目中，本人主要负责医学图像处理与路径规划相关工作，具体包括：

1. 医学图像分割与预处理：基于人体血管硅胶模型断层扫描获得的CT数据，开展血管区域的自动分割研究。通过对医学影像数据的去噪、增强及结构提取，实现血管区域的有效分离，为后续三维建模提供高质量输入。

2. 血管三维模型重建：基于血管分割后的二维切片数据，采用体绘制与表面重建方法重建血管的三维结构，构建具有真实解剖结构特征的三维血管模型。该过程将二维的血管CT图像转化为三维的血管模型，直观呈现了复杂血管的拓扑结构，为路径规划与导航提供实现基础。

3. 血管中心线提取：针对血管路径规划需求，开展中心线提取算法研究，将复杂的血管三维结构抽象为树状曲线线段。保留其血拓扑关系与路径连通性，为后续算法提供结构化输入。

4. 基于中心线的导丝路径规划：在血管中心线基础上，设计术前导丝路径规划算法。通过图搜索与优化策略，生成从入口到目标病灶区域的最优路径，综合考虑路径长度、弯曲程度及安全性约束，为智能导航提供决策依据。

通过上述研究工作，本人掌握了医学图像处理、三维重建及路径规划的关键技术流程，并为构建智能化血管介入机器人系统提供了重要技术支撑。

### 介入手术机器人关键技术与工程实验室建设项目

该项目由深圳市发展和改革委员会支持，围绕介入手术机器人系统的关键技术研发与工程化应用，建设多模态医学影像导航系统、血管介入机器人系统及相关感知与控制平台，推动医疗机器人技术的临床转化与产业化应用。

在该项目中，本人曾参与其中的复杂骨折复位手术机器人相关研究，主要围绕骨骼数据采集建模与路径规划可视化开展工作，具体包括：

1. 医学数据采集与建模
协助胫骨医学影像数据的采集与处理，基于CT数据完成二维到三维的骨骼模型重建，建立具有解剖结构特征的胫骨三维模型。

2. 骨折断面建模与数据生成
设计并实现骨折断面随机生成算法，通过对骨骼模型进行随机切割与扰动，构建多样化骨折断面数据集，用于模拟不同类型骨折场景，为后续算法训练与验证提供数据支持。

3. 骨折复位操作路径可视化
基于三维骨骼模型与断裂结构，与已有的骨折复位操作路径信息。开发相应的可视化系统，实现复位过程的动态展示与交互分析。

4. 手术过程辅助可视化系统开发
结合图形渲染与交互技术，实现骨折复位过程的三维可视化表达，提升手术规划的直观性与可操作性，为人机协同手术提供支持。

通过参与该项目，本人深入理解了多模态医学影像在手术导航中的应用，并掌握了从数据建模到路径规划再到系统可视化的完整技术链路。
